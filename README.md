# di-factorycontainer-imageexplorer-android
Image Explorer App - Dependency Injection with Factory Container through a Singleton Class Wiring access. Display movies using the "themoviedb" api.

# Instructions
To run the app in debug please put an api_key (development key) valid for "themoviedb" api consumption in root project `keys.properties` file.

```
API_KEY_DEVELOPMENT = your_development_key
...
```
